# Mi primera App para Android

Espectativas de la aplicación:
- La app consta de 2 tabs.
- Debe consumir la api de noticias https://newsapi.org
- En el primer tab se debe mostrar un listado con los resultados de dicha api.
-  Al tocar una noticia debe abrir el link en un webview.
- Se debe permitir marcar las noticias favoritas del listado, las cuales deben aparecer en el segundo tab.
- Las noticias marcadas como favoritas deben persistir (SQlite)
